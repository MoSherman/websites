British Museum Map

The British Museum Map is a searchable mysql backed database designed to help users find rooms and objects within the museum. Based on a bootstrap front-end and a WSGI 'DIY' Python webframework the project is designed to be used seemlesly on either a desktop or mobile device. The British Museum's own availble map is not designed for simple searching like the BMM project. For users who wish to find specific rooms or objects within the museum in a simple search engine the BMM project is ideal.
