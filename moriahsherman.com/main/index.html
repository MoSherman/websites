British Museum Map

The British Museum Map is a searchable mysql backed database designed to help users find rooms and objects within the museum. Based on a bootstrap front-end and a WSGI 'DIY' Python web framework the project is designed to be used seamlessly on either a desktop or mobile device. The British Museum's own available map is not designed for simple searching like the BMM project. For users who wish to find specific rooms or objects within the museum in a simple search engine the BMM project is ideal.

The Maze of The Assassins

This simple text based game can be played either in your terminal or your browser. The main purpose of this game was to teach myself a bit more about Python logic and decision making trees, particularly what part loops play in automating tasks. It was made in the same vein as game's like Zork. This project uses a Bootstrap template to be accessible to both desktop and mobile users. Future iterations of the project will hopefully include many more rooms, images of the scenes, a 'decision' tracker, and improved aesthetics.
